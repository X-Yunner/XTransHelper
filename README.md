# X翻译助手

一款支持鼠标穿透，窗口置顶，自定义背景颜色，字体颜色，字体大小的翻译软件。

需要大家自行提供相应平台的id和key

目前支持的平台

1.百度翻译

2.火山翻译

3.腾讯翻译

推荐使用腾讯翻译，因为免费额度最高。
